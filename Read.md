# Advanced BMI Calculator

## Overview

The Advanced BMI Calculator is a Python desktop application developed using Tkinter, PostgreSQL, and Matplotlib. It allows users to calculate BMI, store records in a database, view historical data, and analyze BMI trends through graphs.

## Features

* BMI Calculation
* BMI Category Classification
* PostgreSQL Database Storage
* Historical Record Tracking
* BMI Trend Analysis
* Input Validation
* Error Handling
* User-Friendly GUI

## Technologies Used

* Python
* Tkinter
* PostgreSQL
* psycopg2
* Matplotlib

## BMI Categories

| BMI Range    | Category      |
| ------------ | ------------- |
| Below 18.5   | Underweight   |
| 18.5 - 24.9  | Normal Weight |
| 25.0 - 29.9  | Overweight    |
| 30 and Above | Obese         |

## Installation

1. Install Python.
2. Install required libraries:
   pip install psycopg2-binary matplotlib
3. Create the PostgreSQL database and table.
4. Run BMI_Calculator.py.

## Creativity and Problem-Solving

This project extends a basic BMI calculator by integrating PostgreSQL for permanent data storage and Matplotlib for graphical BMI trend analysis. Input validation and exception handling improve reliability and user experience.
